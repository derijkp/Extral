Filing commands

dirglob dir pattern
returns a list of all filenames in directory dir mathing the given pattern
lload filename
returns all lines in the specified files as a list
lwrite file list
writes a list to a file
readfile filename
returns the contents of the file given by filename
writefile filename data
create a file by the name filename with data as its content
getcomplete channelId
get one complete Tcl command, in the sense of having no unclosed quotes, braces, brackets or array element names, from the open file given by channelId
splitcomplete data
split data into complete Tcl commands in the sense of having no unclosed quotes, braces, brackets or array element names.
parsecommand line
parses a cmdline. it returns a list where each element is the part of the cmdline that will result in one element when the cmdline would be evaluated. eg. % parsecommand {set test [format "%2.2f" 4.3]} set test {[format "%2.2f" 4.3]}
cload filename
returns the contents of the given file als a list of complete Tcl commands.
